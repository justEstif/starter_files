# TypeScript Express Project

1. `npx create-mf-app`
  - create src folder ->
  - create app.ts, server.ts, routes/index.ts
2. `npx tsc --init`
  - edit tsconfig
    ```json
    "outDir": "./dist",
    "rootDir": "./src",
    ```
2. `npm init @eslint/config`
3. `npm i ts-node-dev -D`
	- `"dev": "tsnd --respawn ./src/server.ts"`
